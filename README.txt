EEG_Clustering Plugin for EEGLAB
================================

Version: 1.0.0  
Author: Komal K. Kukkar (University of Houston; email: kkkukkar@cougarnet.uh.edu)  
Supported MATLAB Versions: R2024a, R2025a  
Required EEGLAB Version: 2025.0.0  

Overview
--------
The EEG_Clustering plugin is a MATLAB-based extension for EEGLAB that facilitates group-level EEG analysis.  
It enables users to create and cluster EEGLAB STUDY files for data visualization and statistical comparisons  
across conditions or subject groups. The plugin streamlines workflows for preparing data, plotting, and  
performing cluster/source-level analysis.

Key Features
------------
- Group-combined and group-level data preparation for further analysis.
- Cluster/source-level plotting and statistical analysis.
- Automated organization of results and outputs.
- User-friendly integration into the EEGLAB GUI.

Requirements
------------
- MATLAB R2024a or R2025a
- EEGLAB 2025.0.0 or later

Installation
------------
1. Download and extract the EEG_Clustering plugin into the EEGLAB "plugins" folder.
2. Open EEGLAB.
3. Navigate to **Tools → EEG Clustering** to launch the plugin.
